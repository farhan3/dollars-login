The Dollars' login page from Durarara!!
---------------------------------------
####Requirements

You will need [Node.js](https://nodejs.org/).

####How to run

    # clean the project
    $ make clean
    
    # download the necessary dependencies  
    $ make build

    # run the app
    $ node app.js
